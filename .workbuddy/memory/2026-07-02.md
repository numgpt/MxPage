## 修复 "一键生成" 90% 报错

- 问题：`generateAllSections` 的 `try...finally` 缺少 `catch` 块，第10个模块 `fetch` 网络错误（`TypeError: Failed to fetch`）直接崩溃到 React 错误边界
- 修复：`components/planner/planner-workspace.tsx`：
  1. 在 `for` 循环内部给每个 `fetch` 请求包上 `try...catch`，网络错误时标记模块为 FAILED，继续下一个
  2. 外层 `try` 也加上 `catch` 兜底，避免任何意外异常导致白屏
  3. `finally` 保持 `setBulkGenerating(false)` 确保按钮状态恢复
- 深层原因：AI 图片生成请求耗时长，第10个模块可能超时或网络中断，前端需要容错
